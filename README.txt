# AdGame
A test build of code for learning how to write a game.

Download, extract and run the City of Thieves.exe inside the main folder.

This is a debug version and will remain as such until the game itself is complete.

Basic instructions for the game are as follows:
Press [Enter] to send commands and to step to the next page during the introduction.
At any decision process use [#] to select the option you want.
